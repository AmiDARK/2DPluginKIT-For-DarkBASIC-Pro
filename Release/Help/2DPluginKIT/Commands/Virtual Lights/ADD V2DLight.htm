ADD V2DLIGHT
This command will create a new virtual 2D lights.
X and Y are the position where the light will be inserted inside the layer LAYERID.
RGBColor is the light color in RGB24bits
Range# is the range of the light (radius)
Intensity control the amount of light availabls from 0% (=no light) to 100% (=full bright light)
LightEFFECT control how the light will be rendered:
- LightEFFECT = 1 : The light will use cast normal mapping on tiles where normal map texture is defined.
- LightEFFECT = 2 : The light will cast shadow through tiles where mask texture is defined.
- LightEFFECT = 3 : the light will use effect 1 and 2

SYNTAX
NewLIGHTID = ADD V2DLight( X, Y, RGBColor, Range#, Intensity%, LightEFFECT, LayerID )

ADD PARTICLE2D
That command will create a new particle set.
Amount = Define how many objects will be used to create the particle set.
Texture = Which image will be used to texture the particles.
Size = Define the size of each particles in the set.
IDParticle = Number of the particle set that will be defined and returned by the function.

SYNTAX
IDParticle = Add Particle2D( Amount, Texture, Size )

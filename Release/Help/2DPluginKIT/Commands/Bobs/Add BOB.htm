ADD BOB
This command will create a new BOB. BOBS are in fact sprites but to differentiate them from default DarkBASIC Professional sprites, we decided to name them BOB so, the Sprites used with 2DPlugKIT are BOBS.
If you specify FramesOnX and FramesOnY, This mean that your bob is animated and all animations are on the same image.

 SYNTAX
ReturnVALUE = Add Bob( IMAGEID [,FramesOnX, FramesOnY] )

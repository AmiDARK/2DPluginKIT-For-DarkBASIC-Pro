COLLISIONS / PIXELS


2DPluginKIT does not own like eXtends a complete perfect pixel collision. Because each kind of 2D games need a different method to handle collisions that can be more/less complex and mode/less accurate, we decided to directly include fast commands to directly get the color, mask and nmap pixel from a layer. With this, the user can create its own custom pixel collision without the need to know which tile is in the layer. The system will automatically do the job and return the exact pixel, mask, nmap color.

COLLISIONS COMMANDS :

Get Layer MASKPixel
Get Layer NMAPPixel
 Get Layer Pixel

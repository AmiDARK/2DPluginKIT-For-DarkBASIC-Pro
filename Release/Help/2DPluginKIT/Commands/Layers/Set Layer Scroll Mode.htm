SET LAYER SCROLL MODE
This command is linked to work with SET GAME LAYER and SET GAME LAYER POSITION.
It defines how this layer will be scrolled depending on Game Layer
Mode 0 = Layer does not scroll. It is always drawn at offset 0, 0
Mode 1 = Layer scroll is relative to game layer. Set Layer Scroll Speed XY commands will define its speed regarding to game layer one.
Mode 2 = Constant. This layer will scroll constantly using Set Layer Scroll Speed XY defined values.

SYNTAX
Set Layer Scroll Mode LAYERID, MODE

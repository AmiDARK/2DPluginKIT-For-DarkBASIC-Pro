INTRODUCTION & INSTALLATION

 


Welcome to the DarkBASIC Professional "2DPluginKIT" plugin Pack help system.
An essential addition to the programming language
DarkBASIC Professional.


The main objectived of this pack is to make the 2D games development easier by providing various support like tiles, layers, 2D particles, lights.

!!! WARNING !!!
The most important thing to know is that you must have the latest version of PurePLUGIN installed in the Compiler/Plugins-user/ folder otherwise 2DPluginKIT will not work. PurePLUGIN.DLL version 1.4.0.1 is available in the 2DPluginKIT package. As a remind, PurePLUGIN.DLL is free and can be downloaded from Odyssey-Creators website.

Copy both 2DPluginKIT.dll and PurePLUGIN.dll from Compiler\Plugins-User\ folder,
to your DarkBASIC Professional's Compiler\Plugins-User\ folder

You'll get the following error message : "DBProExtends\GetDBProBase init failed". if PurePLUGIN.DLL 1.4.0.1 or newer is not detected.

Second thig to know if that you must insert your User/Code in the samples provided before trying to compile them otherwise the plugin will fail to initialize and your programs will not work.
